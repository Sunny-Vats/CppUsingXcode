# Spiral Linked List
- Problem Statement: Given a Linked List, the traversal of linked list should start from middle element and print linked list in spiral form.
- Ex:
```text
    1 --> 3 --> 5 --> 7 --> 9 --> 14 --> NULL
                      |
                     mid
    Output:
    7 --> 5 --> 9 --> 3 --> 14 --> 1 --> NULL
```
